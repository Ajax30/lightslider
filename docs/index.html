2. How to use the plugin ↑

Basic usage:

1. Include the latest (or simply, your favorite) version of the jQuery library right after the closing tag in your html document:

<script type="text/javascript" src="https://code.jquery.com/jquery-1.11.3.min.js"></script>

2. Bellow that line, add the jquery.lightslider.min.js file:

<script type="text/javascript" src="js/jquery.lightslider.min.js"></script> 3. Include the plugin's stylesheet: <link rel="stylesheet" type="text/css" href="css/lightslider.css">

4. Make un unordered list of linked images and, if you want, captions.

5. Attach the plugin to the container of both the navigation and the content.

<script type="text/javascript">  $(function() {    $('ul.slider').lightSlider();  }); </script">

3. Options ↑

Option Type Value Role
auto boolean The default value is "true". If set to "true", the slides autoadvance
arrowsNavigation boolean The default value is "true". If set to "true", the Next and Previous buttons are visible
bullets boolean The default value is "true". If set to "true", the Next bullets are visible
loop boolean The default value is "true". If set to "true", the slides loop
slideDuration integer The default value is 500 The duration, in miliseconds, of the transition between two slides
pause integer The default value is 4000 The pause, in miliseconds, between two slide transitions
